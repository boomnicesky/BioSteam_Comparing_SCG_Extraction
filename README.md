# TEA & LCA of Spent Coffee Grounds (SCG) Extraction using BioSteam

## 📌 Overview
This repository contains a BioSteam-based simulation for the extraction of bioactive compounds from Spent Coffee Grounds (SCG). The study compares **Ethanol-based extraction** against **Hexane-based extraction** to evaluate economic viability and environmental sustainability.

## 🚀 Objectives
- To simulate full-scale industrial extraction processes for SCG.
- To perform **Techno-Economic Analysis (TEA)** including IRR, NPV, and MPSP calculations.
- To conduct **Life Cycle Assessment (LCA)** focusing on GWP,CED, Water,and HT.
- To optimize solvent recovery efficiency and its impact on the baseline IRR.
